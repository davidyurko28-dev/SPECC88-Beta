    How to write a great Agent Prompt

    Bau deinen Prompt wie ein Mini‑Briefing. Nutze dieses Format:

    { "goal": "What should the agent achieve?", "inputs": ["What data will the user provide?"], "rules": ["Constraints, tone, brand voice"], "outputs": ["What should be produced (form, emails, page)"] }

    Shortcut: Wenn du unsicher bist, bitte kurz ChatGPT, dir so einen Prompt zu schreiben. Formuliere nur dein Ziel & Kontext – kopiere das Ergebnis hier rein. Fertig.
